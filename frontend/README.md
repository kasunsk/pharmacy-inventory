# Frontend Module (React + Vite)

This module provides inventory list and detail views for pharmacy staff.

## Setup and Run

```powershell
Set-Location "C:\Users\kasun\OneDrive\Desktop\Projects\pharmacy-inventory\frontend"
npm install
npm run dev
```

## API Configuration

Frontend uses `VITE_API_BASE_URL` or defaults to `http://localhost:8080`.

Create `.env` in this module if needed:

```env
VITE_API_BASE_URL=http://localhost:8080
```
