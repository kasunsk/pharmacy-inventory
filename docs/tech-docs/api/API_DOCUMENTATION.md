# API Documentation

This section points to key API areas used by the application.

## Main Endpoints

| Area | Base Path |
| --- | --- |
| Auth | `/auth` |
| Inventory | `/inventory` |
| Sales | `/sales` |
| User/Profile | `/employees`, `/profile` |
| Docs | `/docs/tech` |

## Response Expectations

- Successful responses return JSON payloads unless explicitly requested as plain text.
- Validation and operational errors include an HTTP status and message.
